# <img src="https://raw.githubusercontent.com/zhg7/melonseed/61a9b91e191b38e279fe4c1dde164e025df7898b/app/assets/images/logo/melon.svg" width="42px"></img> Melonseed
Escaparate virtual de frutas en el que se hace uso de las APIs de Wikipedia y OpenStreetMap.

> [!NOTE]
> No data is stored, as there is no backend. Permissions requested and their purposes:
> * Microphone: to use the voice search.
> * Location: to display user's approximate location (OpenStreetMap privacy policy applies).
> * Notifications: to show a notification upon sign-in or sign-up.
> * Cookies: to store a randomly generated UUID in user's browser.
